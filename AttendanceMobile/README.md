# 📱 Attendance Mobile Node

[![Platform](https://img.shields.io/badge/Platform-React%20Native%20%7C%20Expo-blue.svg?style=for-the-badge)](https://expo.dev)
[![Feature](https://img.shields.io/badge/Feature-QR%20Scanner%20Bridge-green.svg?style=for-the-badge)](https://github.com/markDelin/Attendance_System)

A dedicated mobile companion for the **QR Tools Attendance System**. This app acts as a secure "Mobile Node" that bridges your device's camera hardware directly to the central PHP attendance server, bypassing common browser-based camera restrictions.

---

## 🚀 Quick Setup

### 1. Install Dependencies
Ensure you have Node.js installed, then run:
```bash
npm install
```

### 2. Launch the Node
Start the Expo development server:
```bash
npx expo start
```

### 3. Connect to Server
1. Open the app on your physical device (via Expo Go or a built APK).
2. Look at the **Main Dashboard** on your PC to find your **Local IP Address** (e.g., `192.168.1.5`).
3. Enter the IP into the Mobile Node setup screen.
4. Tap **Connect to LAN Server**.

---

## 🛠️ Key Features

- **Hardware Camera Bridge**: Optimized QR scanning performance via native WebView.
- **Persistent Configuration**: Remembers your server IP across app restarts using `AsyncStorage`.
- **Easy Reset**: Long-press the floating button at the bottom for 2 seconds to reset the IP configuration.
- **Network Optimization**: Auto-detects connection errors and provides visual feedback.

---

## 📖 Deployment Guide

### Development
To test on your device, download the **Expo Go** app from the App Store or Play Store and scan the QR code generated by `npx expo start`.

### Production Build
To create a standalone Android/iOS app:
```bash
npx eas build --platform android --profile preview
```

---

## ⚠️ Troubleshooting

- **Connection Failed**: Ensure both the PC and the Mobile device are on the **same Wi-Fi network**.
- **Firewall Block**: On Windows, ensure your firewall allows traffic on **Port 8000**.
- **Camera Not Loading**: Check if you have granted Camera Permissions to the app in your device settings.
